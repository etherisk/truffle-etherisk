Welcome to EthRisk aka Kindarisky.

In order to play this game you will:

Download the latest version of Mist https://github.com/ethereum/mist Right now, you need to install the latest version and run it manually to make it work. Please follow the instructions at https://github.com/ethereum/mist#development

You need to switch on the Test network. The game has not been yet deployed on the real network. You need an account with some ETH. To get ETH on the test network, you need to use a ETH faucet like at https://zerogox.com/ethereum/wei_faucet

Mist is a browser, so you need to enter the URL to the game in Mist. Then you need to link your account to the current page.

Then you can either join a game or create a new one.

When enough players have joined, you can start the game!
